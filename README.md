# StackLogix Demo

This is the interactive StackLogix dashboard page, integrated with an Express backend and PostgreSQL database.

## Architecture
- **Frontend**: React, Framer Motion, Tailwind/Vanilla CSS, Lucide icons.
- **Backend**: Node.js Express server.
- **Database**: AWS RDS PostgreSQL database.
- **Deployment**: Single container Docker SDK on Hugging Face Spaces.
